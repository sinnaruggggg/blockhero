서버 연결

Supabase와 관리자 권한을 입력하는 영역입니다. 드래프트 저장, 최신 배포 불러오기, 배포 이력 조회가 여기서 시작됩니다.

기기 미리보기

화면 종류와 휴대폰 해상도를 고릅니다. 안전영역까지 포함해서 실제 폰 비율에 가깝게 미리보기를 맞추는 메뉴입니다.

배경 설정

레벨 월드별, 특정 레벨별, 레이드 단계별 배경을 바꾸는 메뉴입니다. 이미지 키, 틴트, 제거 여부를 설정하고 바로 무대에 반영됩니다.

배포 메모 / 이력

이번 배포에서 무엇을 바꿨는지 적고, 이전 배포본을 다시 불러오거나 롤백 배포를 만드는 메뉴입니다.
